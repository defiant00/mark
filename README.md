* `_text_` _italics_
* `*` unordered list
* `n.` ordered list
* `---` divider
* `# text` to `##### text` heading
* `> text` quote
* `[ ]` unchecked check box
* `[X]` checked check box
* `\n` is a newline
  * multiple `\n` is a new paragraph
