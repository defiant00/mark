# mark

* `*text*` **bold**
* `_text_` _italics_
* `*` unordered list
* `n.` ordered list
* `---` divider
* `` `code` `` inline code
* ` ```code``` ` code block
* `# text` to `##### text` heading
* `> text` quote
* `[ ]` unchecked check box
* `[X]` checked check box
* `\n` is a newline
  * multiple `\n` is a new paragraph
* link
* image
* anchor
